# rock-paper-scissors
### **Project Description**

This project is a **Rock, Paper, Scissors** game developed in **Python**. It is a console-based application where the user plays against the computer. The computer randomly selects one of the three options—Rock, Paper, or Scissors—using Python's `random` module, while the user enters their choice through the keyboard.

The game compares both choices according to the standard rules of Rock, Paper, and Scissors, determines the winner, and updates the scores after each round. It continues to run until the user types **"End"**, after which the final scores of both the player and the computer are displayed.

This project demonstrates the use of Python concepts such as **loops**, **conditional statements**, **lists**, **user input**, **random number generation**, and **score tracking**, making it an excellent beginner-level project for learning Python programming.
